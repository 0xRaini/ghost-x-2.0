# Ghost X 图标生成说明

## 生成图标步骤

1. **打开图标生成器**
   - 双击 `generate_ghost_icons.html` 文件
   - 或在浏览器中打开该文件

2. **下载图标文件**
   - 点击"下载 icon16.png"按钮
   - 点击"下载 icon48.png"按钮  
   - 点击"下载 icon128.png"按钮

3. **保存到正确位置**
   - 将下载的3个PNG文件保存到 `images/` 文件夹中
   - 确保文件名正确：`icon16.png`, `icon48.png`, `icon128.png`

4. **重新加载扩展**
   - 在Chrome扩展管理页面重新加载Ghost X扩展
   - 扩展将显示新的幽灵图标

## 图标设计

- **背景**：Twitter蓝色 (#1DA1F2)
- **幽灵**：白色幽灵形状
- **眼睛**：蓝色圆点
- **嘴巴**：蓝色弧形线条
- **风格**：简洁、现代、符合Twitter设计语言

## 图标用途

- **16x16**：浏览器工具栏图标
- **48x48**：扩展管理页面图标
- **128x128**：Chrome商店图标

## 注意事项

- 确保图标文件保存在 `images/` 目录中
- 文件名必须与manifest.json中的配置一致
- 图标应该是PNG格式，支持透明背景
